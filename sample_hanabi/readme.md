##　説明
花火は「中心からランダムな方向に飛ぶ」必要があります。
方向をランダムに決めるのがこれ：
angle = random.uniform(0, 2 * math.pi)


① まず cos と sin って何？

cos（コサイン）と sin（サイン）は三角関数です。
「角度から、横と縦の割合を求める関数」です。

■ 円で考えるとわかりやすい

半径1の円を考えます。

角度 θ の方向に線を引くと、
その先の点の座標はこうなります：

x = cos(θ)
y = sin(θ)


つまり：

cos → 横方向の成分

sin → 縦方向の成分

を教えてくれます。

② なぜ花火で cos と sin を使うの？

花火は「中心からランダムな方向に飛ぶ」必要があります。

方向をランダムに決めるのがこれ：

angle = random.uniform(0, 2 * math.pi)

これは 0度〜360度（0〜2πラジアン） の間のランダムな角度を決めています。

③ 角度だけでは動けない

角度だけでは「どの方向か」はわかりますが、

横にどれだけ動くか？

縦にどれだけ動くか？

はまだ分かりません。

そこで：

横の割合 → cos(angle)
縦の割合 → sin(angle)


を使います。

④ speed を掛ける理由

cos と sin は「割合」です。

例えば：

cos(θ) = 0.8

sin(θ) = 0.6

これは

横に 80%

縦に 60%

の割合で進むという意味。

そこに

* speed


をかけることで、

「実際の速さ」に変換しています。

⑤ つまりこの2行は何をしている？
self.vx = math.cos(angle) * speed
self.vy = math.sin(angle) * speed


意味は：

「ランダムな方向に、決められた速さで飛ぶための
横方向の速さ（vx）と縦方向の速さ（vy）を計算している」

⑥ イメージ図

角度 θ が 45° のとき：

    ↑ y
    |
    |   *
    |  /
    | /
----+------→ x


この斜めの線を
横と縦に分解しているのが
cos → 横成分
sin → 縦成分
です。


---
① まず大前提：Pythonの角度は「ラジアン」
math.cos() や math.sin() は 度（°）ではなくラジアンという単位を使います。

② ラジアンって何？
円の「半径」を基準にした角度の単位です。

■ 円周の長さを思い出す

円の円周は：

2 × π × 半径


半径が 1 の円（単位円）なら：

円周 = 2π


つまり、

円をぐるっと一周すると長さは 2π

になります。

③ ラジアンの定義

ラジアンはこう定義されます：

「弧の長さ ÷ 半径」

半径1の円なら、

弧の長さ ＝ 角度（ラジアン）

になります。

だから：

半周 → π

1周 → 2π

になります。

④ 度との対応
| 度（°） | ラジアン |
| ---- | ---- |
| 0°   | 0    |
| 90°  | π/2  |
| 180° | π    |
| 360° | 2π   |
