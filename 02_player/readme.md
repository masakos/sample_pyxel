
## ジャンプの仕組み
上向きの速度を与える（マイナス）
毎フレーム重力を足す
地面に戻ったら止める

ジャンプは実は「上に飛ばして、重力で落としている」だけです。

✔ 上向きの速度 = yを小さくするマイナスの値
✔ 重力を足す = 毎フレーム下向きの力を加える
✔ 地面で止める = 位置と速度をリセットする
