
🎮 基本ルール

あなたは 画面下の三角形（プレイヤー） を操作します。
上から落ちてくる 四角い敵 を撃ち落としてください。

敵に当たるとゲームオーバーです。

🕹 操作方法

| キー    | 動作            |
| ----- | ------------- |
| ← →   | 左右移動          |
| SPACE | 弾を発射          |
| R     | ゲームオーバー後リスタート |

 
🎯 目的

弾を当てると +10点

できるだけ多くの敵を倒してスコアを伸ばす


💥 ゲームの流れ

画面下に自機（三角）がいる
上から敵（四角）が落ちてくる
SPACEで弾を撃つ
弾が敵に当たると敵が消える
敵が自機に当たると「GAME OVER」


🔥 面白くなる改造例

- 今のバージョンは「入門版」です。
- 改造すると一気にゲームっぽくなります：
- 連射可能にする
- 敵のスピードを徐々に上げる
- 爆発エフェクト追加
- ライフ制にする（3回当たるまでOK）

##  コードの説明
pyxel.run(self.update, self.draw) は 「ゲームを動かし続けるスイッチ」 です。　

ゲームはこう動いている
ゲームは実はずっとこれを繰り返しています：
- 状態を更新する（update）
- 画面を描き直す（draw）
- すぐまた①に戻る
これを 1秒間に約60回 くり返しています。
だから動いて見えます。

pyxel.run() は 「update と draw を何度も何度も実行し続ける仕組み」
